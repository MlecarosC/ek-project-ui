Sobre el Proyecto

Martin Lecaros
Desarrollador Full Stack | Eureka 2025

Sistema integral de gestión de candidatos desarrollado con arquitectura de microservicios, diseñado para optimizar y modernizar los procesos de reclutamiento y selección de personal.

El proyecto implementa las mejores prácticas de desarrollo de software, incluyendo arquitectura escalable, tests automatizados end-to-end (Jest para frontend y Testcontainers para backend) y contenerización con Docker.

Frontend:
Angular 20 Tailwind CSS DaisyUI Jest
Backend:
Spring Boot 3.5 MySQL Testcontainers JUnit
Arquitectura:
Microservicios Eureka Server API Gateway Docker
